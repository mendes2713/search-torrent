# searchtorrent
 Simple torrent search on multiple sites simultaneously (idope.se, btdb.to, torrentz)

## Description
 Uses html scraper api for search on multiple sources and return a list on JSON array object.

### Install

```
npm install searchtorrent
```

### Search magnet links 
```javascript
var torrent = require('searchtorrent');
torrent.search('Ubuntu').then(function(data){
   console.log(data); 
});
